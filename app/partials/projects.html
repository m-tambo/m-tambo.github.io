JuxtaPosition

Side-by-side comparison tool for fantasy football enthusiasts.

This application allows a user to input any two NFL players and see a full report of their projected stats and rankings for a given week. A front-end web app built using Javascript frameworks AngularJS and jQuery. Materialize provided a slick minimalist styling option. Several API calls made to nfl.com and fantasyfootballnerd.com.

Ensemblist

Bringing professional freelance musicians together.

This application seeks to connect musicians by allowing them to create a gig and invite nearby pros to collaborate with them. A full-stack web app built with NodeJS, ExpressJS, AngularJS and Bootstrap. For a database PostgreSQL was used, along with the Knex/Bookshelf ORM.
